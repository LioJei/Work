# Changelog

此项目的所有重要变更都将会在这个文档中列出。

## [Unreleased]

## [0.1.5] - 2023-01-11

### Added

- 添加修复 macOS 下提示应用已损坏的文档
- 多线程转换
- 用 GitHub Action 构建 Windows 和 macOS 的可执行文件。
- 支持 Windows 32bit

### Changed

- 将下载镜像切换到 ghproxy.com
- 用 Qt5 重构以支持 Windows 7 和 Windows 8
- 更新 caj2pdf 子模块

## [0.1.4] - 2022-04-22

### Added

- 文件多选
- 文件拖拽

### Fixed

- 用 QProcess::execute 来同步执行命令

### Removed

- 删除 Qt6::Core5Compat

## [0.1.3] - 2021-11-27

### Fix

- 修复图片转换问题。

## [0.1.2] - 2021-11-24

### Fix

- 修复中文路径问题。

## [0.1.1] - 2021-11-20

### Added

- 完成网站。

## [0.1.0] - 2021-11-20

### Added

- 完成初始版本。
